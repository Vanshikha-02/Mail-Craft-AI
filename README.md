## 📖 About

MailCraft AI is a modern AI-powered writing assistant designed to simplify professional communication.

Using the **Groq API**, it generates natural, well-structured, and professional content in just a few seconds.

Whether you're applying for a job, requesting leave, sending a follow-up email, or writing a business proposal, MailCraft AI helps you write faster and better.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Vanshikha-02/Mail-Craft-AI.git
```

Go into the project

```bash
cd Mail-Craft-AI
```

Install dependencies

```bash
npm install
```

Create a `.env.local` file

```env
GROQ_API_KEY=your_groq_api_key
```

Run the development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```
